pub fn parse_file(
    file_content: &str, 
    file_id: SourceFileId
) -> Result<(Module, Vec<Diagnostic>), Vec<Diagnostic>>

Parse a Module from the file content string.

If there was no fatal error during parsing, it returns the parsed module and a vector of Diagnostics, which should be printed. If any of the returned diagnostics are errors, the compilation of this file should ultimately fail.

If the parser does reach a fatal error, this returns the list of generated diagnostics.

A SourceFileId is required to associate any diagnostics with the underlying file.
